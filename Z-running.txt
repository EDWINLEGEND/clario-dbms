# Clario Project - Quick Start Guide

## Prerequisites
- Docker Desktop running
- Node.js installed

## 1. Start Database & Services
cd infra
docker-compose up -d
cd ..

## 2. Setup Backend
cd backend
npm install
npx prisma generate
npx prisma migrate dev
npx prisma db seed
npm start
cd ..

## 3. Setup Frontend
cd frontend
npm install
npm run dev
cd ..

## Access Points
- Frontend: http://localhost:3000
- Backend API: http://localhost:4000
- Database Admin: npx prisma studio (from backend folder)
- MailHog: http://localhost:8025
- n8n: http://localhost:5679

## Quick Commands
# Seed database only
cd backend && npx prisma db seed

# Reset database
cd backend && npx prisma migrate reset

# View logs
cd infra && docker-compose logs
