## **Primary UI Framework & Design System**

### **Component Library (Post-Refactoring)**
- **shadcn/ui** - Exclusive component library providing Cards, Buttons, Inputs, Avatars, Progress, etc.
- **Framer Motion** - Advanced animation system with spring physics, stagger effects, and accessibility support
- **Tailwind CSS** - Utility-first CSS framework with custom interactive classes
- **Lucide React** - Unified icon system across all components

### **Design Philosophy**
- **Unified Dark Theme**: Premium black background with white/opacity text hierarchy
- **Interactive-First**: Every element responds with glow, glaze, and lift effects
- **Accessibility-First**: Built-in reduced motion support and WCAG AA compliance
- **Mobile-First Responsive**: Adaptive layouts with touch-friendly interactions

## **Key UI Style Elements**

### **Visual Design**
- **Dark Theme Foundation**: `bg-black` with `text-white` hierarchy
- **Interactive Glow Effects**: `.interactive-glow` class with `box-shadow: 0 0 15px 3px rgba(255, 255, 255, 0.1)`
- **Glazing Effects**: `.btn-glaze-hover` with mouse-tracking radial gradients
- **Card Lift Animations**: `hover:-translate-y-2` with smooth 300ms transitions
- **Glassmorphism**: `backdrop-blur-xl` with `bg-white/5` overlays

### **Color System (Unified)**
- **Background**: `bg-black` (primary), `bg-white/5` (subtle), `bg-white/10` (cards)
- **Text Hierarchy**: 
  - `text-white` (primary)
  - `text-white/80` (secondary)
  - `text-white/60` (muted)
  - `text-white/50` (disabled)
- **Borders**: `border-white/10` (standard), `border-white/20` (emphasis)
- **Interactive**: `hover:bg-white/10` (subtle), `hover:bg-white/90` (buttons)

### **Typography**
- **System Fonts**: Inter, system-ui, sans-serif
- **Weight Scale**: 300-900 with semantic usage
- **Size Scale**: xs to 4xl with consistent line heights
- **Dark Theme Optimized**: High contrast ratios for accessibility

### **Animation & Interactions**

#### **Entrance Animations**
- **Stagger Container**: `staggerChildren: 0.1` for sequential reveals
- **Spring Physics**: `type: "spring", stiffness: 300, damping: 30`
- **Item Variants**: `y: 20, opacity: 0` ‚Üí `y: 0, opacity: 1`
- **Card Entrance**: Scale + fade + slide combinations

#### **Hover Interactions**
- **Glow Effect**: White glow appears on hover (300ms transition)
- **Lift Animation**: Cards lift 4-8px with easing
- **Glazing Effect**: Mouse-tracking radial gradient

#### **Accessibility Features**

## **Advanced Interactive Effects**

### **Glow System**
```css
.interactive-glow {
  box-shadow: 0 0 0px 0px rgba(255, 255, 255, 0.2);
  transition: box-shadow 0.3s ease-in-out;
}

.interactive-glow:hover {
  box-shadow: 0 0 15px 3px rgba(255, 255, 255, 0.1);
}
```

### **Glazing Effect**
```css
.btn-glaze-hover {
  background: radial-gradient(
    circle at var(--x, 50%) var(--y, 50%),
    rgba(255, 255, 255, 0.1) 0%,
    transparent 50%
  );
}
```

### **Animation Patterns**
```typescript
// Stagger Container
const containerVariants = {
  hidden: { opacity: 0 },
  visible: {
    opacity: 1,
    transition: { staggerChildren: 0.1 }
  }
};

// Item Animation
const itemVariants = {
  hidden: { y: 20, opacity: 0 },
  visible: { 
    y: 0, 
    opacity: 1,
    transition: {
      type: "spring",
      stiffness: 300,
      damping: 30
    }
  }
};
```
